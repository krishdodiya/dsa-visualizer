# DSA Visualizer

A small interactive tool to understand how common algorithms work step by step.
Instead of just looking at code, you can see how things change as the algorithm runs.

---

## What it does

* Visualizes algorithms step by step
* Shows variables, states, and transitions
* Lets you control execution (play, pause, step)
* Includes explanations to make things easier to follow

---

## Supported Algorithms

**Arrays & Hashing**

* Two Sum

**Sliding Window**

* Longest Substring Without Repeating Characters
* Maximum Sum Subarray

**Searching**

* Binary Search

**Linked List**

* Reverse Linked List

**Graphs**

* BFS (Breadth-First Search)
* DFS
* Dijkstra’s Algorithm

**Stack**

* Valid Parentheses

---

## Features

* Step-by-step execution
* Learning + Debug modes
* Graph and array visualizations
* Pseudocode alongside execution
* Adjustable speed controls

---

## Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS

---

## Getting Started

Clone the repo:

```bash
git clone https://github.com/your-username/dsa-visualizer.git
cd dsa-visualizer
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

## Why I built this

I wanted something that makes algorithms easier to understand visually instead of just reading solutions. This started as a personal learning tool and grew into something more structured.
